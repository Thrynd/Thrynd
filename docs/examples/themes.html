How to Use

Switch themes by changing the body class:

<body class="theme-modern">  <!-- Modern theme -->
<body class="theme-classic"> <!-- Classic theme -->
<body class="theme-minimal"> <!-- Minimal theme -->
<body class="theme-neon">    <!-- Neon theme -->
<body class="theme-nature">  <!-- Nature theme -->
<body class="theme-corporate"> <!-- Corporate theme -->

Or use JavaScript to switch dynamically:

document.body.className = 'theme-neon';
